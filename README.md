## tipos de dados

Conforme o ECMAScript standard temos 9  tipos de dados:

* Data types

* Primitive / primitive value
* Structural 
* structural primitive

## Primitivos

* string "ABC" "BCA" "bca" "Bca" //não sofrem alterações
* number
* boolean
* undefined 
* symbol
* BigInt

## Estruturais

* Object 
* Array
* Map
* Set
* Date
* ...

* Funtion
